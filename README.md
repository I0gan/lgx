# lgx web server

## Complie lgx server

You should make sure you have install complie env as follows:
### Debian, Ubuntu, Kali ....

```
sudo apt install g++
sudo apt install make

```
### Get open source
First, you should git clone src to complie

```sh
git clone https://github.com/I0gan/lgx.git
cd lgx
```
Now you can comple with make
### comple
```
make
```

### run

```
chmod +x start && ./start

```
